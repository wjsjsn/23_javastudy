-class Ex01{}

1. 클래스 이름=저장명
2. 클래스 이름의 첫글자는 무조건 대문자
3. 클래스 : 자바에서 저장할 수 있는 최소 단위
4. 클래스 = method&field로 구성(하나만 있어도 상관 없지만 클래스 안에 있어야 함)
5. field(상태값=데이터, 아는 것) => 변수(변할 수 있는 상태값)&상수(변하지 않는 상태값)

-public static void main(String[] args){}

1. () = method(동작, 하는 일) => 모든 프로그램의 main method는 무조건 1개, 제일 위에 있어야 함(프로그램의 시작점)
2. ()앞에는 method 이름
3. main method = JVM이 호출해서 사용하는 method

-System.out.println("자바의 꿈);

1. System.out => 모니터(표준출력장치)
2. System.in => 키보드(표준입력장치)
3. println("내용") => 내용 출력
